# Bike Sales Data Analysis (Excel Project)

## Project Overview
This project analyzes customer bike purchase behavior using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Data Cleaning

## Business Questions Solved
1. Average income of customers who purchased bikes
2. Gender vs bike purchases
3. Age group vs bike purchases
4. Commute distance vs bike purchases
5. Region vs bike purchases
6. Education level vs bike purchases
7. Occupation vs bike purchases
8. Home ownership vs bike purchases

## Dashboard
The dashboard uses slicers and pivot charts to allow interactive analysis.

## Skills Demonstrated
- Data Cleaning
- Pivot Tables
- Data Visualization
- Business Insight Generation

## conclusion
The ideal bike buyer profile is a middle-aged professional in North America with a short commute (0–2 miles) and a slightly above-average income. Gender and homeownership have little predictive power. Marketing efforts should be concentrated on the middle-age segment and short-commute urban areas, while the Pacific region and senior/young age groups represent under-served markets worth exploring.
